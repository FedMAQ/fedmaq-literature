# Agent Memory

Durable facts about this repo's literature review — paper inventory, OKF node
state, conversion status — live in this repo's git-tracked files: `kg/`
(curated OKF nodes), `markdown/` (raw converted papers), `.claude/project/paper_registry.md`,
and `.claude/project/changelog.md`. These are authoritative.

claude-mem is for session recall only (what was discussed, why a paper was
tagged a certain way) — not a substitute for reading current `kg/` or registry
state. Before acting on a claude-mem recollection that names a specific paper,
node, or registry status, verify it still matches the current `kg/` content —
node claims have been revised before to match manuscript de-overclaiming.
