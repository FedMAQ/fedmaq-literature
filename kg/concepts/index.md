# Concepts

`type: Concept` documents for cross-cutting ideas that span many papers —
quantization, knowledge distillation, non-IID data / client drift, communication
efficiency, differential privacy, edge/IoT deployment.

_Scaffolded; not yet populated._ This layer is Phase 2 of the OKF restructure.
Concept nodes will define each idea once and link out to the [papers](/papers/)
and [methods](/methods/) that use it, so shared terminology in the manuscript
resolves to a single authoritative node. The `tags` already present on paper
nodes (`quantization`, `kd`, `heterogeneity`, `survey`, `application`, …) seed the
initial concept set.
