# Gaps

`type: Gap` documents recording open problems in the literature that FedMAQ
targets — each gap describing what remains unresolved, which papers surface it,
and how FedMAQ's multi-adaptive quantization + knowledge distillation addresses it.

_Scaffolded; not yet populated._ This layer is Phase 3 of the OKF restructure.
Gaps will be distilled from the limitations and "gap addressed by FedMAQ" notes
already present in the [paper nodes](/papers/) — e.g. static vs. adaptive
bit-width, quantization under non-IID drift, the proxy-dataset requirement of
distillation methods — each linked to the [papers](/papers/) that motivate it and
the [findings](/findings/) that bear on it.
